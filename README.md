## Описание проекта

Проект представляет собой интеграцию трёх desktop-модулей:
- **Модуль "Технолог"**
- **Модуль "Лаборатория"**
- **Модуль "Аппаратчик"**

Модули взаимодействуют с общей базой данных и API. Разработаны в рамках учебной практики по профессиональному модулю ПМ.02 "Осуществление интеграции программных модулей".

## Состав репозитория

| Папка | Содержание |
|-------|------------|
| `docs/` | Все документы практики (отчёт, дневник, аттестационный лист, пояснительная записка, руководство оператора, отладка модуля, презентация) в форматах .docx и .pdf |
| `database/` | Скрипт создания и заполнения базы данных (SQL) |
| `json/` | Коллекция Postman для тестирования API |
| `txt/` | Ссылка на программу |

---

## Как запустить проект

### 1. Развертывание базы данных
- Запустите скрипт `database/script.sql` в вашей СУБД (PostgreSQL или MS SQL Server).
- Убедитесь, что созданы таблицы и загружены тестовые данные.

### 2. Запуск API
- Импортируйте коллекцию `ZAM_API_Complete.json` в Postman.
- API доступен по адресу: `http://localhost:65163` (при локальном запуске).

### 3. Запуск desktop-модулей
- Откройте решение `desktop-modules/TechModule/TechModule.sln` в Visual Studio.
- Соберите и запустите проект. Аналогично для `LabModule` и `OperatorModule`.
